Programming Languages

A programming language is a formal language, which comprises a set of instructions that produce various kinds of output. Programming languages are used in computer programming to implement algorithms.

Examples:

  1. C++
  2. Java
  3. Python
  4. C#
  5. R

Hyper Text Markup Language

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML is not a programming language!
